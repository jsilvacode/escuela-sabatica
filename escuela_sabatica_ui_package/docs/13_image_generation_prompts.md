# 13 — Prompts para generar mockups visuales

Estos prompts sirven para recrear o extender las imágenes de referencia en herramientas de generación visual. Pueden usarse en Figma AI, Midjourney, DALL·E, Ideogram, Galileo, Uizard o herramientas similares.

## 1. Design system board

```txt
Create a high-fidelity UI design system concept board for a modern responsive web platform for Adventist Sabbath School study, intended as a Figma-ready visual foundation for later implementation in Codex Desktop. This is not a marketing poster; it is a polished product-design mockup board.

Use a premium editorial web-app style: calm, elegant, spiritual, readable, contemporary, warm, and minimal.

The board should present the visual concept for a site called "Escuela Sabática" integrated with a Bible experience called "Santa Biblia". Show the UI in Spanish.

Include:
- Brand header / logo treatment for "Escuela Sabática".
- Color palette with labeled swatches: deep navy, muted blue, warm ivory, sand/beige, soft gold accent, slate text, muted olive accent.
- Typography guidance with examples using a clean sans-serif for interface and an elegant serif for reading.
- UI tokens / spacing / radius / shadow examples.
- Reusable components: sticky top navbar, left sidebar, hero section, lesson cards, day tabs, mobile day selector, scripture reference chips, scripture modal with tabs "Biblia" and "Comentario", audio player, search bar, resource cards, primary and secondary buttons, footer.
- Small annotations in Spanish: "lectura cómoda", "navegación clara", "referencias bíblicas interactivas", "apto para móvil".
- Thumbnail previews of desktop and mobile screens.

Design language:
- Premium editorial reading app + spiritual study platform + modern web app.
- Soft rounded corners, subtle shadows, generous whitespace, elegant hierarchy, accessible contrast.
- The board should look like a Figma presentation frame or UX handoff moodboard.
```

## 2. Desktop home

```txt
Create a polished high-fidelity desktop web mockup for the homepage of a modern Adventist Sabbath School study platform.

Design a desktop-first homepage in Spanish for "Escuela Sabática" integrated with "Santa Biblia".

Include:
- Sticky top header with brand/logo, navigation "Inicio", "Lecciones", "Recursos", "Santa Biblia", "Plan de lectura", search, profile icons and CTA.
- Left sidebar with Inicio, Lecciones, Recursos, Biblia, Marcadores, Notas, Ajustes.
- Large hero with calm landscape, quarter theme title, subtitle and buttons "Comenzar estudio" and "Ver recursos".
- Verse-of-the-quarter card and progress widget.
- Summary card about the trimester theme.
- Grid of 13 weekly lesson cards with number, title, dates, thumbnail and button "Leer lección".
- Resource cards for PDF, audio, presentations and material for teachers.
- Footer.

Use warm ivory background, deep navy, muted blue, soft gold CTA and olive progress accents. Make it realistic, implementable and Figma-ready.
```

## 3. Weekly lesson desktop

```txt
Create a polished high-fidelity desktop web mockup for the weekly lesson detail page.

Include:
- Sticky top header and left sidebar.
- Large lesson header card with banner image, lesson number, title, dates, short intro and actions "Continuar lección" and "Escuchar audio".
- Horizontal day tabs: Sábado, Domingo, Lunes, Martes, Miércoles, Jueves, Viernes, with active state.
- Main reading preview area for active day with day title, excerpt and scripture references as clickable chips.
- Right column with widgets: "Versículo clave", "Progreso semanal", "Notas rápidas", "Recursos de la semana".
- Related resources cards below.

Spanish UI, premium editorial spiritual web-app style.
```

## 4. Daily reading desktop with Bible modal

```txt
Create a polished high-fidelity desktop web mockup for the daily reading page.

Include:
- Sticky top header and left sidebar.
- Day label, day title, breadcrumb and header image.
- Compact selector of week days with current day active.
- Main article column with generous whitespace, serif reading text, headings, highlighted key verse block and inline scripture references such as "Génesis 1:1" and "Juan 3:16" styled as interactive links.
- Embedded audio player near the top.
- Right sidebar with progress, bookmarks, notes and study actions.
- Open scripture modal layered over the page with reference title, tabs "Biblia" and "Comentario", verse text, and buttons/icons for copiar, compartir, marcar, destacar and abrir en Biblia.

The UI must clearly show that the user can study the lesson and instantly open Bible/commentary without leaving the page.
```
